# dotfiles

Files for my developing environment.

## requirements

- zsh
- prezto
- ghq
- peco
- powerlebvel10k

## usage

1. exec `setup.zsh`
2. `echo "zstyle :prezto:module:prompt theme powerlevel10k" >> ~/.zpreztorc`
3. `p10k configure`
4. append some options from `zprofile` to `~/.zprofile`
* .vimrc
