# dotfiles

Files for my developing environment.

## requirements

- zsh
- sheldon
- ghq
- peco
- starship

## usage

1. exec `./setup.zsh`
2. `sheldon lock` (downloads all plugins)

-----

# Installation

- [windows](https://qiita.com/mtsgi/items/8a844870f30b30ef21e4)

## install zsh

```sh
# windows
sudo apt install zsh
zsh
# press q
./setup.zsh
```
